# Weitere Aktionen

## Anmeldesystem ändern

Mit dem **Stift**-Symbol können Sie bestehendes Anmeldesystem ändern.

Die Eingaben sind die selben wie beim [Anlegen eines neuen Anmeldesystems](/docs/anmeldetool/anmeldetool-admins/neues_anmeldesystem.md).

## Anmeldesystem löschen

Mit dem **Papierkorb**-Symbol können Sie ein bestehendes Anmeldesystem löschen.

Ein Anmeldesystem kann nur dann gelöscht werden, wenn es leer ist.

## Anmeldesystem ein- oder ausschalten

Mit dem **Ein-/Ausschalten**-Symbol können Sie ein bestehendes Anmeldesystem ein- bzw. ausschalten.

Nur bei eingeschalteten Anmeldesystemen können sich Benutzer anmelden.
