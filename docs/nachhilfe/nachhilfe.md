# Nachhilfetool

Die Schülerinnen des Christian-Doppler-Gymnasium haben eine Aktion **"Schüler helfen Schülerinnen"** ins Leben gerufen.<br />
Es geht darum, dass erfahrene Schüler:innen anderen Schüler:innen ihre Hilfe anbieten.

Eine perfekte Idee, die durch eine Online-Version unterstützt gehört.

Genau diese Online-Version wird aktuell entwickelt.

![Nachhilfetool](@site/static/img/undraw_video-tutorial.svg)

## Status

Die Online-Version ist derzeit **in Entwicklung**.<br/>
Geplant ist die Veröffentlichung mit dem 2. Semester 2025/26.

### Aktuelle Lage

- Schüler:innen können sich am System anmelden (2.12.2025)
